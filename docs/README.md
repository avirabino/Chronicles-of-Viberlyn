# Chronicles of Viberlyn Documentation

This directory contains all reference materials and documentation for the Chronicles of Viberlyn project.

## Directory Structure

- `game-design/` - Game design documents, mechanics, and systems
- `lore/` - World building, story, and narrative content
- `technical/` - Technical documentation and implementation details
- `ai/` - AI integration documentation and research
- `rules/` - Game rules and mechanics documentation

## Contributing to Documentation

When adding new documentation:
1. Place files in the appropriate subdirectory
2. Update this README with any new sections
3. Follow the established markdown formatting
4. Include relevant metadata (date, version, author) in the document header

## Documentation Standards

- Use Markdown for all documentation files
- Include a table of contents for longer documents
- Keep documentation up to date with code changes
- Use clear, concise language
- Include examples where appropriate
